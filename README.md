# HR-analytics-dashboard
This repository contains an HR Analytics Dashboard designed to analyze and visualize key workforce metrics with a focus on employee attrition. The dashboard helps HR teams and business leaders gain actionable insights into employee behavior, retention risks, and workforce trends.

Total Employee Count

Average Years of Experience

Department-wise Attrition
